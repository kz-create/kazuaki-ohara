### 株式会社サイバーエージェント（2023年9月～現在）

#### 担当プロダクト概要
- **Retail-X**

### **WALL形式プロジェクトのリアーキテクチャ（2025年　10月　〜　2025年　11月）**

#### プロジェクト概要

- WALL形式ではクライアントメディアごとに個別のUIを表示する場合があった為、共通機能を利用するメディア、個別機能を利用するメディアが混在しており、影響範囲が特定しづらく、開発コストを上げる要因になっていた。そのため、コードベースをTurborepoを使用したモノレポ構成に統合し、各々workspaceで管理することで、スケーラビリティの向上と開発生産性の改善を実現する。

#### 担当フェーズ

- 要件定義、アーキテクチャ設計、実装、テスト

#### 使用技術

- 言語/FW: TypeScript, React, Vite
- モノレポ: Turborepo, pnpm workspace
- インフラ: Amazon S3, CloudFront
- CI/CD: GitHub Actions

#### 実績・取り組み

- apps（各メディア）とpackages（共通的なコンポーネントやロジック）による疎結合なアーキテクチャを設計。
- 11個の共通パッケージ（types, utils, theme, context, core, hooks, features等）を開発。
- Turborepo Generatorsを使用した新規メディア追加の自動化スクリプトを実装。コマンドを実行することで、対話形式でテンプレートに沿った新規メディア用のworkspaceを作成可能。
- 複数環境（dev/stg/prd）へのビルド・デプロイフローを整備。

#### 工夫した点

- 依存方向を「apps → packages」の単方向に制限し、循環依存を防止。
- 新規メディア追加時間を0.5日から5分に短縮し、開発生産性を大幅に向上。
